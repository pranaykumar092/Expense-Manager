# 💰 Expense Manager (SAVE.IT)

A clean, responsive, and easy-to-use **Expense Tracker** built with HTML, CSS, and JavaScript. This application allows users to set a total budget, add various expenses, and keep track of their remaining balance in real-time.

**Live Demo:** [Click Here to view the Live Project](https://pranaykumar092.github.io/Expense-Manager/)

---

## ✨ Features
- **Budget Management**: Set a total budget to keep track of your financial limits.
- **Expense Tracking**: Easily add expenses with titles and costs.
- **Real-Time Calculations**: The app automatically updates your Total Budget, Expenses, and Remaining Balance as you add items.
- **Expense List**: View a detailed list of all recorded expenses.
- **Clean UI**: A sleek, card-based user interface using Google Fonts (Poppins) and FontAwesome icons.
- **Input Validation**: Prevents empty or negative values to ensure accurate tracking.

## 🛠️ Tech Stack
- **HTML5**: Structured markup.
- **CSS3**: For custom styling, flexbox layout, and a modern design.
- **JavaScript (Vanilla)**: Handles logic for budget calculations, expense tracking, and DOM updates.

## 🚀 How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/pranaykumar092/Expense-Manager.git
   ```
2. **Navigate into the directory:**
   ```bash
   cd Expense-Manager
   ```
3. **Open the project:**
   Simply double-click on `index.html` to open it in any modern web browser. No server required!
